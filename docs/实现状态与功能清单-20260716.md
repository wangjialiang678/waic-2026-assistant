## 一、已实现功能（全部上线）

### 数据与内容
官方 210（论坛 174 + 场馆活动 36 `venue_based`）· 展区 4 · 边会 138 · 参展商 1020（25 行业）· 情报 231。每条带来源；无源边会标「待核实」；并入来源存 `additional_sources`。

### 前端浏览
- 四板块 SPA（官方/边会/参展商/情报）+ 顶层导航（浏览·我的日程带badge·人脉对接·AI原住民·接入AI助手）。
- **全局搜索**（搜索时跨官方+边会一起搜）+ **活动权重体系**（超脑=100置顶高亮 > 全体会议 > 展区 > 主题论坛 > 官方合作边会 > 分论坛…）。
- 官方板块：天/片区/场馆/类别（含场馆活动）筛选；场馆下拉只取官方场馆；venue_based 标「场馆内·非官网议程」。边会排序：有日期→有源→待核实。
- **展区导览**：四馆官方介绍+大图+亮点+H1–H4+地址交通+高德链接，四馆各自不同代表照。
- **我的日程**：localStorage 无登录、撞车提醒、导出 .ics、分享长图。
- **展商品牌别名**（24 家，行吟信息=小红书等）：搜品牌能命中 + 卡片标品牌。

### AI 聊天助手
Qwen3.5-Flash（腾讯云 TokenHub）SSE 流式 + function-calling 返回活动卡片。**相关性**：标题/标签精确匹配 + 关键词归一（AI教育→教育）；**去重**：跨轮卡片按 id 去重。后端挂→静态兜底。

**跨天日期匹配（2026-07-16 修复）**：横跨全会期的活动（如超脑四天展区/体验计划、国泰海通论坛）`date` 是区间字符串 `2026-07-17~20`，`day=null`。此前按天筛选用精确比对 → 单日查询（如 7/18）全被漏掉。现 `tools.py` `_act_on_day` / 前端 `app.js` `actMatchesDay` 统一用区间覆盖判断，四天活动在其覆盖的每一天都出现；`search_activities` 排序加 `weight` 次序键，超脑同相关度置顶。

**来源三档标注**：`source_type=official`→「官方」；`source_type=unofficial` 但 `waic_relation=official`（WAIC 官方合作项目，如超脑）→「WAIC 官方合作」；其余 unofficial→「非官方·以官方为准」。避免超脑等官方合作被误标「非官方」。

### 一日行程规划器（旗舰，借鉴 HiWAIC/WaytoAGI）
按天 + 关注方向 / **persona 快选**（投资人/创业者/开发者/学生家长/AI从业）+ 节奏 → 纯客户端确定性算法产**无撞车、少换馆**动线；带**今日时段热力图**（避高峰）；一键全加入日程。

### 无登录跨端同步 + 自动兴趣
匿名**同步码** + 服务端 SQLite（`/api/state`，LWW）→ 多设备 + AI agent skill 间同步日程/兴趣。兴趣 23 项（含 AI教育/一人公司·OPC/智能体·Agent/创作者·Builders）+ 自定义 + **行为自动推断**（加入日程1/浏览详情0.5，≥2 自动升为已关注，「猜你关注」建议）。

### 人脉对接（社交速配 M9）
> 前端展示名统一为「人脉对接」（商务场合，去相亲/娱乐味）；内部代号 M9、类名 `soc-*` 不变。
匿名兴趣匹配（名片/发现/我的匹配三页）+ **显式同意** + 自愿联系方式 + **双向 like 才互露联系方式** + 一键删除 + kill-switch（WAIC_SOCIAL）+ 按同步码限流。**入口不再自动弹出**（`.soc-root[hidden]` 守卫修复：此前 `display:flex` 盖掉 `hidden` 导致加载即弹、且关不掉，规划器/CMS 同类弹窗一并修复）。

### 展商自助 CMS M11
认领（留资，人工线下核验营业执照）→ 审核（`/cms-admin.html` token）→ 通过后受限编辑简介/官网/补充 → 前端合并标「商家提供」。官方层只读、编辑层独立（每日刷新不冲掉商家编辑）；XSS 剥离 + 官网白名单。

### 一键多端导航 · 周边指南 · 纠错入口
- 活动详情页**一键导航**（高德/百度/Apple/Google）。
- **周边实用指南** `/guide.html`（三地四馆交通/片区接驳/入场须知/住宿餐饮，来源官方）。
- **纠错入口**：详情页「信息有误?报错/纠正」→ `/api/report` 收集 → admin 可查。

### skill 接入（v2.1.0）
bundle 带索引层、详情走 API；**数据自更新**（/VERSION→bundle）+ **skill 自升级**（/SKILL_VERSION→重下 SKILL.md）+ 跨端同步章节（同步码）。

### 推荐 / 导流 / 专题
推荐区（Hi WAIC 官方 App 重点 + WayToAGI 边会）+ 页脚官方渠道 + 超脑@WAIC 专题页（生命力博物馆等）。

### 运维
- **缓存**：nginx `Cache-Control: no-cache`（按 etag 校验，改动即达用户）+ 资源版本号 `?v=`。
- **限流**：nginx per-IP 600r/m（放宽防会场共享 WiFi 误伤）+ app 层**按同步码**每设备限流 + 每 IP 兜底。
- **两层部署** `deploy.sh`；日志 JSONL（IP 哈希脱敏，封顶 ~500M）；社交/CMS/纠错皆 kill-switch / token 保护。
