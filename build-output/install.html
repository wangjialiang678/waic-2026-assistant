接入 AI 助手

把 WAIC 2026 全量日程装进你的 AI（Claude Code / OpenClaw 等），随时问「今天世博中心有哪些大模型论坛」「有没有需要报名的 afterparty」。

它能做什么

174 官网论坛36 场馆活动4 展区138 边会/周边1020 参展商231 资讯
  • 查官方论坛/展区：按天、场馆、片区、类别、主题、嘉宾检索
  • 查边会/周边/社群活动：时间地点、是否需注册、票价、报名链接
  • 查参展商展台：按展馆(H1-H4/西岸/张江)、行业、名称检索
  • 帮你维护个人日程、查时间冲突；每条信息都带官方/非官方来源

安装（一条命令）

把这份「AI 说明书」放进你的 runtime 加载 skills 的目录（各家不同：Claude Code 是 ~/.claude/skills/，OpenClaw 是 ~/.openclaw/workspace/skills/，Codex / Hermes / WorkBuddy 等放各自的 skills 位置）：

SK=<你的skills目录>/waic-2026        # ← 换成你 runtime 的实际路径
mkdir -p "$SK" && curl -fsSL https://waic.sg.superbrain-ai.com/SKILL.md -o "$SK/SKILL.md"

更省事的办法：把上面这段直接发给你的 AI 助手，说「装一下这个 skill」——它知道自己的 skills 目录在哪。装好后问任何 WAIC 问题即可，数据与说明书都会自动保持最新。

数据接口（也可直接用）

  • /VERSION — 数据版本号
  • /bundle.tar.gz — 全量数据包（活动+情报+参展商 md/json）
  • /data/activities.json — 352 条活动（官方 210 + 边会 138 + 展区 4，带来源）
  • /data/exhibitors.json — 1020 家参展商
  • /data/intel.json — 231 篇资讯素材
