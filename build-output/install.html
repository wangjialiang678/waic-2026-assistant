安装（一条命令）

最省事（推荐，手机也行）：把这句话发给你的 AI 助手（OpenClaw / Claude Code / Codex 等）：

帮我装一下 WAIC 参展助手 skill：下载 https://waic.sg.superbrain-ai.com/SKILL.md 放进你加载 skills 的目录（waic-2026/SKILL.md），装好后把会期推送和自检的定时任务也设上。

或者自己动手——放进你的 runtime 加载 skills 的目录（Claude Code 是 ~/.claude/skills/，OpenClaw 是 ~/.openclaw/workspace/skills/，其余 runtime 放各自的 skills 位置）：

SK=<你的skills目录>/waic-2026        # ← 换成你 runtime 的实际路径
mkdir -p "$SK" && curl -fsSL https://waic.sg.superbrain-ai.com/SKILL.md -o "$SK/SKILL.md"

装好后问任何 WAIC 问题即可，数据与说明书都会自动保持最新。
