💬 你在微信里打开的话

  1. 最省事：点右上角 「···」 → 「收藏」（之后在 我 → 收藏 一点就开），或选 「浮窗」把页面钉在微信边缘
  2. 想放桌面：点右上角 「···」 → 「在浏览器打开」（微信内没法添加到桌面）
  3. 跳到浏览器后，按下面 iPhone / Android 的两步操作
  4. 更快的兜底：长按下面这张二维码 → 保存图片到相册，之后扫码就能回来
长按图片 → 保存到相册

🍎 iPhone（Safari 或 Chrome 都行）

  1. 点底部/右上角的 分享按钮（方框加向上箭头 ⬆️）
  2. 往下滑，点 「添加到主屏幕」 → 「添加」

桌面就会出现「WAIC助手」图标，点开直达。

🤖 Android（Chrome / Edge 等）

  1. 点右上角 「⋮」菜单
  2. 点 「添加到主屏幕」（或「安装应用」）→ 确认

部分国产浏览器菜单里叫「发送到桌面」，效果相同。

这只是一个桌面快捷方式，不占空间、不要权限、随时可删。
